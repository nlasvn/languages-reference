Typescript

No Typescript usamos tipagem explícita para ajudar o compilador a encontrar bugs referentes aos tipos das nossas variáveis. Não é necessário colocar a assinatura do tipo quando você declara as variáveis com let ou const, pois o Typescript infere elas automaticamente.
Tipos:

  • string
  • number
  • boolean
  • Array<t> ou t[], exemplo: Array<number> ou number[] // é equivalente a list no Python
  • {number, string, boolean} // isso é um object, é equivalente ao dict no Python.

Declarando variáveis

let name = "Marisa";
let height = 155;
let isMage = true;
let spellcards = ["Master Spark", "Stardust Reverie", "Non-Directional Laser"];
const signature = { "name": "Love Sign \"Master Spark\"" }; //constante

Funções

function calcSquare(num: number): number {
  return num * num;
}

const numbers = [1, 2, 3, 4, 5];
const numbersPlus1 = numbers.map(function(x) { return x + 1 }); //anonymous function, equivalente ao lambda no Python
const numbersPlus2 = numbers.map((x) => x + 2); //arrow functions, são uma anonymous function mais ergonômica
const numbersPlus3 = numbers.map((x) => {
  return x + 3;
}); //arrow function em várias linhas, é possível fazer o mesmo com uma anonymous function.
